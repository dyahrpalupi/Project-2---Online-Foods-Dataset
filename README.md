# Project-2---Online-Foods-Dataset
This project focuses on Exploratory Data Analysis (EDA) of an online food ordering dataset using Python.  The analysis aims to understand customer behavior, demographic patterns, and factors influencing online food ordering decisions.

# Tools-Used:
Python, Pandas, NumPy, Matplotlib, Seaborn

# Analysis Scope
The analysis covers:
• Data structure and missing value inspection
• Basic data cleaning and preprocessing
• Exploratory analysis of customer demographics
• Visualization of key variables and relationships
• Summary of insights based on observed patterns
